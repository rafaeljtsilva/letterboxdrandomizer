Confirma se o teu username está correto e se tens filmes na Watchlist!"); } else { // Como a página já veio baralhada do Letterboxd, // só precisamos de tirar os 3 primeiros que aparecem! sortearFilmesNativos(); } } catch (erro) { console.error(erro); alert("Erro ao aceder ao Letterboxd.
